# 飞牛IPTV

飞牛IPTV 是一款为飞牛私有云（fnOS）开发的 IPTV 直播应用，集成央视频播放源，支持 Web 独立访问和 M3U8 第三方播放器订阅。

## 功能特性

- **央视频道直播**：集成央视频播放源，支持 CCTV 全频道，以及各大卫视等60余个频道
- **独立 Web 访问**：通过 IP+端口直接访问，无需登录飞牛桌面
- **M3U8 订阅**：支持 PotPlayer、Kodi、TiviMate 等第三方播放器
- **自定义频道**：支持导入 M3U 播放列表，管理个人频道
- **EPG 节目单**：显示频道当前节目信息
- **PHP 一键安装**：央视频源通过 PHP 代理获取，支持一键安装/卸载
- **账号密码保护**：独立 Web 页面支持登录认证
- **多线程并发**：支持多客户端同时访问
- **PID 持久化驻存**：服务崩溃后自动重启

## 安装方法

1. 下载 `fnos_iptv_v2.1.3_web.fpk`
2. 在飞牛 OS 应用中心手动安装
3. 安装完成后从桌面打开应用
4. 进入设置页面，启动【独立 Web 服务】
5. 浏览器访问 `http://NAS_IP:8899`

## 项目结构

```
.
├── app/ui/           # Web 前端和 CGI 后端
│   ├── index.html    # 前端页面
│   ├── index.cgi     # CGI 接口
│   ├── server.py     # 独立 Web 服务器
│   ├── ysp.php       # 央视频代理
│   └── images/       # 图标资源
├── cmd/              # 安装/卸载脚本
├── config/           # 应用配置
└── manifest          # 应用清单
```

## 技术栈

- 前端：原生 HTML + CSS + JavaScript
- 后端：Python CGI + Python HTTP Server
- 播放：HLS.js（浏览器端）+ PHP 代理（服务端）
- 平台：飞牛私有云 fnOS
